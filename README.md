ricoooo84.github.io/shottrack/

HOW TO USE:

Select preferred audio and click basketball to start session.

Use music playback control to log makes/misses
Play/Pause = resume/pause session
Next song = make
Prev song = miss

IMPORTANT:

Meant to be used on Safari/Chrome, some SFX doesnt work on other browsers

Device needs to be in a state that allows next/prev input
In my experience, it will still track if my phone is off but will not play the make/miss sounds until turned on. I just keep my phone on at low brightness.
